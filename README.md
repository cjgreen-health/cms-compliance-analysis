# CMS Compliance Analysis

Independent analysis of 418,000+ CMS health inspection citations across 14,600 
nursing facilities nationwide. Explores the operational, staffing, and geographic 
drivers behind regulatory performance.

## 🔍 What This Covers
- State-level citation benchmarking (2024)
- Florida compliance performance vs. national average
- Staffing tiers as a predictor of citation volume
- Correction velocity by deficiency category
- Citation origin: standard surveys vs. complaint investigations

## 📊 Data Source
CMS Health Inspections Dataset — March 2026 Release  
418,000+ citation records · 2020–2025 survey cycles

## 🛠 Built With
Python · Pandas · Matplotlib · Seaborn
